# MediaExpandFlow

Creator partnership agency website for AI, SaaS, fintech, and technology brands.

## Stack
- Pure HTML5 + Vanilla CSS + Vanilla JS (no framework)
- Google Fonts (Inter)
- Deployed on Vercel
- Custom domain: www.mediaexpandflow.com

## Pages
- `index.html` — Home
- `services.html` — Services
- `about.html` — About
- `contact.html` — Contact

## Local Development
```bash
npx serve . -p 3000
```
Open [http://localhost:3000](http://localhost:3000)

## Deploy
Deployed automatically via Vercel on every push to `main`.
